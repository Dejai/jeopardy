Homemade Jeopardy

Use this page to load the home-made Jeopardy game.

Step 1: Enter Name

Step 2: Enter Game Sheet

  1. You should already have a game created in a Google Sheet.
    • It should have the same columns as the game template
  2. Ensure your Google Sheet is published to the web with document type = Tab-separated values (.tsv)
    • You can publish the "Entire Document" as long as you have one sheet or the game sheet is the first sheet.
  3. Click on START GAME.
